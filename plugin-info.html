Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
changesets:add No description.
changesets:help Display help information on changesets-maven-plugin.
Call mvn changesets:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
changesets:init No description.
changesets:prepare Applies all changesets into the changelog and calculates the new version number.

This would normally be the last step in preparing a release PR.

changesets:release No description.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>se.fortnox.changesets</groupId>
          <artifactId>changesets-maven-plugin</artifactId>
          <version>0.1.1-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>se.fortnox.changesets</groupId>
        <artifactId>changesets-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
